## Feature Highlights

### Core UI
- **Custom Cursor** Рђћ velocity-sensitive trail with 6-node physics, per-element variant states (`project`, `pointer`, `terminal`)
- **Smooth Scroll** Рђћ `@studio-freight/lenis` for buttery momentum
- **Scroll Velocity Skew** Рђћ `skewY()` applied to cards and rows based on scroll delta, RAF-throttled
- **Dark / Light Mode** Рђћ CSS-variable swap, persists across sections
- **Spotlight Background** Рђћ radial CSS mask synced to mouse position, reveals animated grid
- **Network Background** Рђћ `<canvas>` particle mesh, mouse-reactive, theme-aware

### Data & Live APIs
- **GitHub Contributions Heatmap** Рђћ 52-week grid from `jogruber.de` proxy, level-coded colors
- **GitHub Activity Syslog** Рђћ real-time push/create/star events as rotating terminal-style ticker
- **Hashnode GraphQL** Рђћ fetches latest 3 posts and total article count live
- **Last Commit Badge** Рђћ polls the SyncWatch repo for the most recent commit message and relative time
- **Visitor Count** Рђћ live counter via `counterapi.dev`

### Interactions
- **Terminal** Рђћ open with `` ` ``, supports `whoami`, `ls projects`, `cat resume`, `contact`, `cat public.key`, `clear`, `exit`, and `rm -rf /`
- **Command Palette** Рђћ `Cmd/Ctrl + K`, searchable actions with arrow-key navigation
- **Section Navigation** Рђћ `J` / `K` to move between sections, `1РђЊ4` to switch tabs
- **Project Modal** Рђћ full case study with problem, solution, architecture diagram, tech decisions, learnings, and metrics
- **3D Card Tilt** Рђћ `perspective` + `rotateX/Y` on mousemove, with radial glare overlay
- **Article Search** Рђћ instant filter across all 14 articles, with empty state

### Easter Eggs
- **Type "samprati"** Рђћ triggers a glitch effect and full-screen identity reveal
- **Click the logo 10├Ќ** Рђћ confetti burst
- **Rage click (5├Ќ rapid)** Рђћ contextual toast
- **`rm -rf /`** in terminal Рђћ DOM wipe animation, followed by BIOS-style reboot sequence
- **End of page** Рђћ "you made it to the end." toast
- **Copy text** Рђћ "sharing is caring" toast with site link
- **Time-aware greeting** Рђћ morning / afternoon / evening / "you're up late"

### Key optimizations

- **Vendor chunk split** Рђћ React + ReactDOM isolated in `vendor.js`, cached separately
- **Local fonts** Рђћ WOFF2 files served from `/public/assets/fonts/`, `font-display: swap`
- **RAF-throttled scroll handlers** Рђћ `requestAnimationFrame` wraps all scroll events
- **Memoized components** Рђћ `React.memo` on `GitHubHeatmap`, `LiveClock`, `Cursor`
- **`useMemo` on heatmap grid** Рђћ 364-cell calculation runs only when data or theme changes
- **IntersectionObserver** for scroll animations Рђћ no scroll listeners, no layout thrash
- **`will-change: transform`** on animated elements Рђћ promotes to compositor layer
- **`terser`** with `drop_console: true` Рђћ zero console output in production
- **No source maps** in production (`sourcemap: false`)
- **Passive event listeners** Рђћ `{ passive: true }` on all mousemove/scroll handlers

## Design Decisions

**Why no UI library?**
Full pixel control. Every hover state, every timing curve, every border radius was deliberate. Copying a component library's aesthetic would have made this feel generic.

**Why no TypeScript?**
Deliberate for this project. The portfolio was built on a 3-day deadline. TypeScript would have added friction without meaningful benefit at this scale. SyncWatch (the sister project) uses TypeScript.

**Why `@studio-freight/lenis` over native smooth scroll?**
Native CSS `scroll-behavior: smooth` doesn't allow momentum control or scroll velocity reads, both of which power the skew effect on cards.

**Why `DOMPurify`?**
The article descriptions from Hashnode's GraphQL API return HTML strings. Rendering them unsanitized would be an XSS vector. DOMPurify strips everything dangerous before React touches it.

**Why local fonts?**
Remote font requests block rendering. Serving WOFF2 from the same origin eliminates the DNS lookup, TCP handshake, and TLS negotiation that would otherwise add 100РђЊ300ms on first load.
